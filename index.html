What is Standard?

Standard is a rotating group of Magic: The Gathering sets. Most sets enter Standard when they're released and drop out about eighteen months later.

A card is Standard if it has been printed or reprinted into a set currently in Standard.

A deck which contains only Standard cards is called a Standard deck.

Rotation patterns

The set rotation pattern for Standard is currently in a two-year transitory phase between the previous set release pattern and the upcoming one.

The new pattern's changes

  • There will be no more core set releases.
  • Block size will be reduced from three sets to two.
  • There will be a total of two blocks per year, up from one.
  • Blocks will stay in Standard for six months less than they do currently.

Effects on Standard

  • Sets will release into Standard at the same rate (four per year).
  • Sets will fall out of Standard twice as often, in groups half the size.
  • The average number of sets in Standard will drop by one.

The transitory phase

The Theros block and 2015 Core Set will drop out together, as in the old rotation pattern.

The Khans of Tarkir block and Magic Origins, although released in the old style, will drop out in the new; Khans of Tarkir and Fate Reforged will drop out together as if they were a two-set block, followed by Dragons of Tarkir and Magic Origins in the same fashion six months later.

After that, all sets will be both released and dropped in the new style.
